# RumanWrites - Blogging Platform

A clean, modern, and fully functional blogging website built with pure **HTML, CSS, and JavaScript**.

### Features

- **User Authentication**: Login & Signup (with localStorage persistence)
- **Create & Publish Posts**: Rich writing experience with title and content
- **Interactive Features**: Like posts and add comments
- **Dynamic Sections**: 
  - Trending This Week (changes on refresh)
  - Category filtering (Technology, Life Lessons, Writing Tips, Personal Growth)
- **Responsive Design**: Fully mobile-friendly with modern UI
- **My Blog Profile**: View all your published stories
- **Realistic UI**: Inspired by Blogger with smooth interactions and modals

### Technologies Used

- HTML5
- CSS3 (Tailwind CSS)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- localStorage for data persistence

### How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
3. No installation required!
